Command Summary Syntax Example
echo echo [message] prints the message, but processes it first printing stuff in shell
date date [option]..[format] prints time, day of week, and date when you need any of those
hostname hostname [option] [file] prints DNS name, and allows you to set system hostname and NIS server name setting hostname, retrieving network info
arch arch [option] prints computer architecture for information about architecture
uname uname [option] prints system information if you need any system information
uptime uptime [option] prints current time, system runtime, # of logged on users, and load averages as a timer and performance status
who who [option] [filename] prints information about logged in users if you want info about other users
yes yes [message] prints message repeatedly until stopped if you want to REALLY make something clear
sleep sleep [number] [unit of time] pauses for a certain amount of time if you need to pause
clear clear clears the terminal if the terminal is cluttered
